# OpenClaw Backup

> 自动备份 OpenClaw 配置，让数据永不丢失

⚠️ **注意**：当前版本仅支持 **Linux/Mac**。Windows 版本正在开发中。

## 安装

### Linux / Mac

```bash
# 克隆到 OpenClaw skills 目录
cd ~/.openclaw/skills
git clone https://github.com/Hi-Jiajun/openclaw-backup.git
```

或从 ClawHub 直接安装：https://clawhub.ai/skills/openclaw-backup-hiliang

## 特性

- 🎯 交互式配置向导
- 🔄 自动备份，保留多个版本
- 🧹 智能清理，自动转移和删除旧备份
- ⚠️ Linux / Mac 可用（Windows 开发中）

## 快速开始

### Linux / Mac

```bash
# 首次配置（交互式向导）
chmod +x scripts/setup.sh
./scripts/setup.sh

# 执行备份
chmod +x scripts/backup.sh
./scripts/backup.sh
```

详细说明请查看 [SKILL.md](SKILL.md)
