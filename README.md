# OpenClaw 公开 Skills 导航

> 这里存放我愿意公开分享的 OpenClaw skills。

## 公开 Skills 列表

### 1. openclaw-backup

> 自动备份 OpenClaw 配置

**功能：**
- 主备份：按主机名和日期分类
- 自建 skills：单独备份
- 自动清理旧备份

**GitHub:** https://github.com/Hi-Jiajun/openclaw-backup

**安装：**
```bash
clawdhub install openclaw-backup-hiliang
```

---

### 2. openclaw-skills-github-sync

> 自动维护 GitHub 和 ClawHub 同步

**功能：**
- 双仓库同步（private + public）
- 大众化处理（脱敏）
- 自动发布到 ClawHub

**GitHub:** https://github.com/Hi-Jiajun/openclaw-skills-github-sync

**安装：**
```bash
clawdhub install openclaw-skills-github-sync-hiliang
```

---

## 安装方式

### 方式1: ClawHub（推荐）
```bash
clawdhub install <skill-name>
```

### 方式2: GitHub
```bash
cd ~/.openclaw/skills
git clone https://github.com/Hi-Jiajun/<repo-name>.git
```
