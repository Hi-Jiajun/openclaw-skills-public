# openclaw-backup

OpenClaw 配置自动备份工具

## 功能

- 主备份：按主机名和日期分类
- 自建 skills：单独备份
- 自动清理旧备份

## 安装

```bash
# ClawHub
clawdhub install openclaw-backup-hiliang

# 或 GitHub
git clone https://github.com/Hi-Jiajun/openclaw-backup.git
```

## 使用

```powershell
powershell -ExecutionPolicy Bypass -File "backup.ps1"
```
