# openclaw-skills-github-sync

OpenClaw GitHub/ClawHub 同步工具

## 功能

- 双仓库同步（private + public）
- 大众化处理（脱敏）
- 自动发布到 ClawHub

## 安装

```bash
# ClawHub
clawdhub install openclaw-skills-github-sync-hiliang

# 或 GitHub
git clone https://github.com/Hi-Jiajun/openclaw-skills-github-sync.git
```

## 使用

```powershell
powershell -ExecutionPolicy Bypass -File "project-maintainer.ps1"
```
